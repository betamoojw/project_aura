# Project Aura v1.1.0

Release date: 2026-02-22

## Highlights
- New integrated web dashboard at `/dashboard` with tabs for Sensors, Charts, Events, Settings, and System.
- OTA firmware update from the web dashboard (`Settings/System -> Update Firmware`) using uploaded `.bin` files.
- Device UI now switches to a dedicated firmware update page during OTA to avoid a "frozen" screen impression.
- OTA flow hardened with upload lifecycle handling (`begin/write/end/abort`) and OTA slot size checks.

## Notes
- OTA update keeps user settings.
- Full web installer flow still supports complete flashing (bootloader, partitions, app, filesystem).
